# Deepshell Chat UI

Minimal web UI for interacting with the Deepshell backend.

## Usage

1. Run your Deepshell backend API (e.g., on http://localhost:8001).  
2. Open `index.html` in a browser.  
3. Type commands and see responses.

## Notes

- Adjust the API URL in `app.js` if your backend runs elsewhere.  
- No dependencies or build steps required.
